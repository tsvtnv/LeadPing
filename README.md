## ✨ What is this?

LeadPing is a free, open-source lead generation tool. Capture, track, and manage leads from your website with a clean dashboard and database-backed storage. Drop it into any project and start collecting leads immediately.

## 🔥 Features

- **📋 Lead Capture** — Form-based lead collection with validation
- **📊 Lead Dashboard** — View, search, and manage all captured leads
- **🗄️ Database Storage** — Prisma ORM with persistent lead storage
- **⚡ Fast** — Vite-powered dev server, instant HMR

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Framework | React + Vite |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Database | Prisma ORM |

## 🚀 Getting Started

```bash
git clone https://github.com/tsvtnv/LeadPing.git
cd LeadPing
npm install
npx prisma generate
npx prisma db push
npm run dev
```
